# F011 - Workspace Flow

Version: 0.1

Status: Complete

Priority: Highest

Owner: ATA Project

---

# 1. Overview

Workspace Flow defines the complete translation workflow of ATA.

It connects all major features into a single user experience.

Every translation task follows this workflow.

---

# 2. Goal

Workspace Flow should:

- Provide a consistent user experience
- Connect all ATA features
- Reduce unnecessary actions
- Keep the workflow simple
- Ensure knowledge is accumulated naturally

---

# 3. Scope (V0.1)

Included

- Mode Selection
- Translation Workspace
- OCR
- Translation Recommendation
- Translation Optimization
- Assets Recommendation
- Workspace History
- Copy

Not Included

- Multi-user collaboration
- Team review dashboard
- Approval workflow
- Analytics

---

# 4. User Flow

Open ATA

↓

Select Mode

↓

Input Source Text
or
Paste Image

↓

OCR (if required)

↓

Display Recognized Text

↓

User Review OCR

↓

Generate Recommendation

↓

Translation Optimization

↓

Display Assets Recommendation

↓

Display Workspace History

↓

User Edit

↓

Copy Translation

↓

Complete

---

# 5. Workflow Rules

Mode selection is mandatory.

Translation recommendations are generated automatically.

OCR review must be completed before translation.

Users make the final decision.

AI never replaces user judgment.

---

# 6. Design Principles

The workflow should feel natural.

The number of user actions should be minimized.

Users should focus on translation rather than operating the system.
