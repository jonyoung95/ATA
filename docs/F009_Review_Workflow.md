# F009 - Review Workflow

Version: 0.1

Status: Complete

Priority: Highest

Owner: ATA Project

---

# 1. Overview

Review Workflow manages how translation knowledge is verified before becoming shared ATA Assets.

The Review process focuses on maintaining translation quality rather than approving individual users.

---

# 2. Goal

Review Workflow should:

- Improve translation quality
- Prevent duplicate knowledge
- Verify terminology consistency
- Build reliable ATA Assets

---

# 3. Scope (V0.1)

Included

- Review Queue
- Review Status
- Reviewer Comments
- Asset Registration

Not Included

- Approval hierarchy
- Reviewer permissions
- Multi-stage approval
- Review assignment

---

# 4. Review Workflow

Translation Completed

↓

Submit to Review

↓

Review

↓

Register as ATA Asset

Review is the only path for creating shared Assets.

---

# 5. Review Queue

Review Queue displays:

- Source text
- Translation
- Mode
- Review status
- Review comments

Items remain editable until review is completed.

---

# 6. Review Policy

Review verifies translation quality.

Review does not evaluate the user.

Knowledge quality is more important than user authority.

V0.1 does not require a dedicated approval role.

Access control should reduce the need for approval-based workflows.

---

# 7. Design Principles

Review exists to improve shared knowledge.

The workflow should remain simple.

Approval processes should not slow down daily work.
